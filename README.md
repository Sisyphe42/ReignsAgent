# ReignsAgent

ReignsAgent is a modular, headless-first project for Reigns-like card simulation and content review.

Phase 1 implements only `@reigns-agent/core`: a pure runtime with factions, card scheduling, game-over checks, and abstract inventory/tag hooks. It intentionally contains no UI, shops, LLM connectors, file import/export, or reviewer simulation code.
